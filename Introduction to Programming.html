5 Basics of the Web

1. The web

The WWW (World Wide Web) is a collection of web pages written in a language called HTML. Web browsers read this documents and then display it to the user.

2. HTML

HTML stands for Hyper Text Markup Language. A HTML document contains text and allows reference to other documents and links to other pages.

3. URLs

Each page has its own unique address, used to connect to other HTML documents.

4. HTTP

A type of communication method for the web browser to make request via the internet to the servers

2. Basic HTML Vocab

1. Most HTML tags require a closing tag with the exception of "void" tags.
2. An element refers to everything withing the opening and closing tags.
3. An attribute or "a" is a property of an HTML element
4. An element can either be "block" or "inline", the difference between both is one can be styled via "CSS" and another does not.
